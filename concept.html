Core Principle

One Scene, One Scroll

The entire wedding invitation is a single vertical scroll. There are no "pages" â€” the guest descends from the night sky to the street puddle, and the invitation content is embedded within the scene. Wireframe box positions adapt to fit the HK concept; only the user flow order is preserved.

HOW THE SCENE CONNECTS

The background is NOT 5 separate images. It is one continuous vertical composition (360Ã—8000px) built from overlapping layers. Each zone's background gradient bleeds into the next â€” navy sky fades into building silhouettes, which fade into mid-level alleys, which fade into neon streets, which fade into wet ground. The parallax layers (clouds, buildings, neon signs, tram) are placed at absolute positions within this continuous canvas. As the user scrolls, they move at different speeds â€” but they're all part of the same scene. The wedding content (text cards, gallery, maps) floats on top with semi-transparent glass cards.

All wedding content (names, dates, location, gallery, accounts) appears in frosted-glass cards â€” backdrop-filter: blur(20px); background: rgba(255,255,255,0.06); border: 1px solid rgba(255,255,255,0.1).
